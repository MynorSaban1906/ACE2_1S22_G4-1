INICIAR SERVIDOR
    npm run dev

========================== NODE SERVER ==========================

CONSTRUIR IMAGEN NODE
    docker build -t arqui2_server:v1 .

CREAR CONTENEDOR NODE
    docker run -p 5000:5000 arqui2_server:v1

========================== MONGO && NODE=========================

CONSTRUIR APLICACION
    sudo docker-compose down && sudo docker-compose up --build

CONECTAR AL CONTENEDOR
    docker exec -it serverNode bash
